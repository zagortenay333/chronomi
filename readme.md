# Chronomi

**All-in-one timer, stopwatch, pomodoro, alarm, todo, time tracker and flashcards app.**

![preview](./data/screenshots/screenshot.png)

## Download

1. [Download the app](https://github.com/zagortenay333/chronomi/tree/main/bin/Chronomi-x86_64.AppImage).
2. Mark the downloaded file as executable.
3. Run the file.

## Compile

1. Install the libraries: `sdl3`, `freetype`, and `harfbuzz`.
2. Run `make release/debug/asan`.
