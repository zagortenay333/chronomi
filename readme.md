# Kronomi

**All-in-one timer, stopwatch, pomodoro, alarm, todo, time tracker and flashcards app**

> [!WARNING]
> This is currently alpha quality software.

![preview](./data/screenshots/screenshot.png)

## Installation

There is no packaged version of the app right now, so you'll have to compile from source.

To compile, install the `gtk4-devel` packages and run `make release/asan/debug`.
